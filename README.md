# versatile

Java implementation of [vers](https://github.com/package-url/purl-spec/blob/version-range-spec/VERSION-RANGE-SPEC.rst),
*a mostly universal version range specifier*

## Versioning Schemes

| Scheme   | Supported |
|:---------|:---------:|
| Alpine   |     ❌     |
| CPAN     |     ❌     |
| Debian   |     ✅     |
| Ruby Gem |     ❌     |
| Generic  |     ✅     |
| Gentoo   |     ❌     |
| Go       |     ❌     |
| Maven    |     ✅     |
| NPM      |     ✅     |
| Nuget    |     ❌     |
| PyPI     |     ❌     |
| RPM      |     ❌     |
